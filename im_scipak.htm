Low-Cost version of the popular GLOBAL LAB Image/2 Imaging Software Package for Scientific and General-Purpose Imaging Applications. GLI/2 Streamline is a version of Data Translation’s GLOBAL LAB Image/2 software that focuses on the easy to use, graphical programming aspect of the full software. GLI/2 Streamline contains many of the same powerful functions and features. No need for code writing or programming knowledge !

Comparison Chart

Key Features

  • New ! The 3.0 Version of GLI/2 adds three new tools, additional colour processing support, automatic thresholding new analysis capabilities, tool enhancements, and more !
  • GLI/2 can be used at three different levels: As an out-of-the-box, stand-alone scientific imaging application with pre-defined tools; As a base for the addition of user-written custom tools and developing complete applications in visual C++
  • A complete library of 28 tools is provided for measurement, analysis and image processing tasks.
  • GLI/2 supports a wide variety of Data Translation frame grabbers, allowing you to perfectly match the frame grabber type to your imaging requirements
